# Free Assets Directory

This directory contains free images available to all users (including free tier).

## Required Images

Place the following placeholder images here:

1. `image-1.jpg` - Generic portfolio/project image
2. `image-2.jpg` - Generic portfolio/project image  
3. `image-3.jpg` - Generic portfolio/project image
4. `image-4.jpg` - Generic portfolio/project image
5. `image-5.jpg` - Generic portfolio/project image
6. `placeholder.jpg` - Fallback placeholder image

## Usage

These images are used in the `EditableImage` component's image picker for free tier users. Paid users can upload their own images via Supabase Storage.

## Image Recommendations

- Size: 1200x1200px recommended
- Format: JPG or PNG
- Content: Generic, professional stock photos suitable for portfolios
- License: Must be free to use commercially (e.g., Unsplash, Pexels)

You can download free images from:
- https://unsplash.com
- https://pexels.com
- https://pixabay.com
